🧇 X'wafffle Quick Start Guide

Modern Cafe Ordering System

🚀 Getting Started in 5 Minutes

Step 1: Open the Website

Simply double-click index.html in your file explorer, or open it in your browser.

✅ No installation needed! No server required!

Step 2: Set Your WhatsApp Number

Open js/whatsapp.js and find line 42:

const restaurantPhone = '1234567890'; // Update this!

Replace with your number (digits only). Examples:

  • USA: 14155552671
  • UK: 442071838750
  • India: 919987654321

Step 3: Test the Flow

  1. Click "Explore Menu" button
  2. Add some items to cart
  3. Click cart icon (🛒)
  4. Proceed to checkout
  5. Fill in your details
  6. Click "Confirm Order & Pay on WhatsApp"
✅ Success! WhatsApp will open with your formatted order ready to send. That's it!

🎨 Customization

Change Restaurant Name

Replace "FoodHub" with your restaurant name in all HTML files:

<span class="logo-text">Your Restaurant Name</span>

Change Colors

Edit CSS variables in css/style.css:

:root {
    --primary-color: #ff6b6b;    /* Main red */
    --secondary-color: #4ecdc4;  /* Teal accent */
    --accent-color: #ffe66d;     /* Yellow accent */
}

Add Menu Items

Edit MENU_ITEMS array in menu.html:

{
    id: 18,
    name: 'Pizza Margherita',
    price: 12.99,
    category: 'pizzas',
    emoji: '🍕',
    description: 'Classic pizza with fresh tomato'
}

Change Tax Rate

In cart.html and checkout.html, find:

const tax = subtotal * 0.10;  // 10% tax

Change 0.10 to your rate (e.g., 0.15 for 15%)

Add Delivery Fee

Add this in checkout totals calculation:

const deliveryFee = 5.00;
const total = subtotal + tax - discount + deliveryFee;

⚙️ Available JavaScript Functions

Cart Functions (js/cart.js)

  • getCart() - Get current cart array
  • addToCart(item) - Add item to cart
  • removeFromCart(itemId) - Remove item from cart
  • updateItemQuantity(itemId, qty) - Update item quantity
  • clearCart() - Empty entire cart
  • getCartCount() - Get total items in cart
  • getCartTotal() - Get total cart price
  • updateCartCount() - Update cart badge on page

WhatsApp Functions (js/whatsapp.js)

  • generateWhatsAppMessage(cart, name, phone, address, notes) - Create formatted message
  • getWhatsAppLink(message) - Get wa.me link with message
  • sendOrderViaWhatsApp(...) - Open WhatsApp with order
  • logOrderToSession(...) - Save order to session
  • getOrderHistory() - Retrieve past orders

App Functions (js/app.js)

  • showToast(message) - Show notification popup
  • formatCurrency(amount) - Format as $XX.XX
  • isValidPhone(phone) - Validate phone number
  • isValidEmail(email) - Validate email
  • isMobileDevice() - Check if mobile browser
  • printReceipt(...) - Generate receipt

🎟️ Coupon Management

Coupons are managed in cart.html. Current codes:

const COUPON_CODES = {
    'SAVE10': 0.10,   // 10% discount
    'SAVE20': 0.20    // 20% discount
};

To add more coupons, add them to the object:

const COUPON_CODES = {
    'SAVE10': 0.10,
    'SAVE20': 0.20,
    'SUMMER': 0.15,   // Add new coupon
    'VIP30': 0.30     // 30% for VIPs
};

🔧 Troubleshooting

❓ WhatsApp Link Not Opening?

  • Check that WhatsApp is installed on your device
  • Verify phone number format in whatsapp.js (digits only, no +)
  • On mobile, ensure WhatsApp app is installed
  • On desktop, ensure WhatsApp Web is logged in

❓ Cart Not Saving?

  • Check if localStorage is enabled in browser
  • Try a different browser
  • Clear browser cache and refresh
  • Check browser console for errors (F12)

❓ Items Not Showing?

  • Open browser console (F12) and check for errors
  • Verify HTML file paths are correct
  • Try refreshing the page (Ctrl+R)
  • Check that all CSS and JS files are in correct folders
